In order to run this project you need:

 ``` sh

 ```

### Setup
Now setup Node.js along with.json file, run;
``` sh
nmp init -y
```
After initiating npm, you need to install linters by following command;
``` sh
npm install --save-dev hint@7.x
```
After that make .hintrc file and add following code into it;
``` sh
{
  "connector": {
    "name": "local",
    "options": {
      "pattern": ["**", "!.git/**", "!node_modules/**"]
    }
  },
  "extends": ["development"],
  "formatters": ["stylish"],
  "hints": [
    "button-type",
    "disown-opener",
    "html-checker",
    "meta-charset-utf-8",
    "meta-viewport",
    "no-inline-styles:error"
  ]
}
```
